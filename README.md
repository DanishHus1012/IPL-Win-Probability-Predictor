# IPL-Win-Probability-Predictor
PL Win Probability Predictor  Overview  This project predicts the winning probability of IPL teams during a live match using Machine Learning. The model is trained on historical IPL match and ball-by-ball delivery data.
Features

* Data preprocessing and cleaning
* Feature engineering
* Match win probability prediction
* Machine Learning pipeline using Scikit-Learn
* Model serialization using Pickle

Dataset

The project uses:

* matches.csv
* deliveries.csv

Dataset source: Kaggle IPL Dataset

Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Google Colab

Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Probability Prediction
6. Model Export using Pickle

Output

The model estimates the probability of either team winning based on current match conditions such as:

* Current score
* Overs completed
* Wickets remaining
* Required run rate
* Current run rate
